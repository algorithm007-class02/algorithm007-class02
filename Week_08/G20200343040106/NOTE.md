# 学习笔记

## 位运算
x^0 = x
x^1s = ~x
x^(~x) = 1s
x^x = 0
经常用到的位运算：
1. 讲x最右边的n位清零： x&(~0<<n)
2. 获取x的第n位值： (x>>n)&1
3. 获取x的第n位的幂值： x&(1<<n)
4. 仅将第n位置1： x|(1<<n)
5. 仅将第n位置0： x&(~(1<<n))

实战必会：
1. x%2==1  equals (x&1)==1
2. x%2==0  equals (x&1)==0
3. x>>2    equals x/2
4. x=x&(x-1) equals 清零最低位的1
5. x&-x    equals   得到最低位的1
6. x&~x=0
